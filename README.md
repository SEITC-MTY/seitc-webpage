# Webpage for the Student Society for Information Technology and Computing Engineering(SEITC)

## Getting Started

How to run:

Make sure to have dependencies installed:
```bash
npm install
```

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

To build:

```bash
npm run build
```

To checke lint:

```bash
npm run lint
```

## Git commands

Fetch the latest changes from the remote repository:

```bash
git fetch
```

Pull the latest changes from the remote repository:

```bash
git pull
```

Switch to your desired branch (e.g., test):

```bash
git checkout test
```

Push a new change

```bash
git add .
git commit -m "Your commit message"
git push
```
